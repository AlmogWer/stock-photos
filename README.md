<h1 align="center">Stock-photos</h1>

## Project Description

React component to generate stock photos (with search) applied with infinite scroll

## Built with

- HTML
- CSS
- React

# npm-start

[npm start](https://docs.npmjs.com/cli/v7/commands/npm-start)

```
npm install
npm install react-icons --save
```

```
npm-start -g

```
